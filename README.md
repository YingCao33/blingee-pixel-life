# 8.99 Blingee Pixel Life

一个可以进入 2000s / Blingee / 像素风场景的多人网页小世界。玩家可以选择场景、移动角色、收集生活碎片、靠近发光物件互动、聊天，并给自己的角色换装。

## 功能

- 多个怀旧像素场景
- WebSocket 多人同步
- 玩家移动、靠近互动、聊天气泡
- 角色换装：发型、上衣、下装、配饰
- 每个场景自适配的合成 BGM
- 收集、聊天、换装、传送等音效
- 局域网试玩和公网临时隧道试玩

## 本地启动

需要 Node.js 18 或以上版本。

```bash
npm start
```

打开：

```text
http://localhost:8991
```

也可以双击 Windows 启动脚本：

```text
start-multiplayer.bat
```

## 同一网络多人试玩

同一个 Wi-Fi / 热点下，其他人可以打开：

```text
http://你的电脑IP:8991
```

例如：

```text
http://172.20.10.3:8991
```

## 临时公网试玩

先保持本地多人服务器运行，再打开公网隧道：

```text
start-public-tunnel.bat
```

终端里会生成一个类似下面的地址：

```text
https://xxxxx.lhr.life
```

把这个地址发给朋友即可。这个免费地址是临时的，关闭隧道窗口后会失效。

## 正式部署

这是一个无数据库的 Node.js 项目。部署平台需要支持：

- Node.js 18+
- HTTP
- WebSocket
- `PORT` 环境变量

启动命令：

```bash
npm start
```

部署成功后，平台会给出一个稳定公网地址，所有人都可以通过这个地址进入同一个多人世界。
